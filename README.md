# 🚀 DSA 100-Day Interview Preparation Challenge

This repository documents my **100-day Data Structures & Algorithms (DSA) journey**, where I consistently solve curated problems to build strong problem-solving skills and prepare for technical interviews.

* 📅 **Duration:** 100 Days
* 🧠 **Daily Goal:** 3 Problems
